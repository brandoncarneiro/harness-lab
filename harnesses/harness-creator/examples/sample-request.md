# Sample Request

I want to create a harness for software delivery.

Today I describe a coding task in ChatGPT, ask a CTO-style model to make a better prompt, paste that into Claude Code, then paste the diff into Codex for review. I want that process structured so every step has an artifact and the final result gets audited.

I do not want the harness to merge code or deploy anything. It can recommend a branch name, but it should ask before creating branches or pushing code.
